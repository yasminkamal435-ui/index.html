deep learning

Core concepts I've been studying.

Neural Networks & Backpropagation
How a network learns: forward pass to make a prediction, then backpropagation to adjust weights based on the error, layer by layer.
Convolutional Neural Networks (CNNs)
Architectures built for images — using convolutional filters to detect edges, shapes, and patterns, which is why they power most computer vision work, including my YOLO project.
Recurrent Networks & LSTMs
Architectures designed for sequences — text, time series, audio — that carry memory of previous steps to understand context over time.
Activation Functions
Functions like ReLU and Sigmoid that introduce non-linearity, letting networks learn complex patterns instead of just straight lines.
Loss Functions & Optimization
How "wrongness" is measured (loss) and reduced over time using optimizers like SGD and Adam to gradually improve the model.
Regularization (Dropout, Batch Norm)
Techniques that prevent a model from memorizing training data too closely, so it generalizes better to new, unseen data.
Transfer Learning
Reusing a model already trained on a large dataset and fine-tuning it for a new, smaller task — faster and often more accurate than training from scratch.
Attention & Transformers
The architecture behind modern LLMs — letting a model weigh which parts of the input matter most when making each prediction, the foundation of the RAG work I do.
